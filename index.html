Enseñando Estructura HTML

Esta página mantiene su estructura básica, pero hemos aplicado CSS al navbar para mostrarlo horizontalmente.

Tipos de Listas en HTML

1. Lista No Ordenada (ul)

Se usa cuando el orden de los elementos no es relevante. Nota que esta lista sí conserva los puntos porque no tiene estilos.

  • Pan
  • Leche
  • Huevos
  • Frutas
    • Manzana
    • Plátano

2. Lista Ordenada (ol)

Se usa para pasos secuenciales o rankings.

  1. Abrir el editor de código
  2. Crear un archivo .html
  3. Escribir la estructura básica
  4. Abrir en el navegador

3. Lista de Descripción (dl)

Ideal para glosarios o diccionarios.

HTML
Lenguaje de marcado para la estructura de la web.
Navegador
Software que interpreta el código HTML y lo muestra visualmente.
